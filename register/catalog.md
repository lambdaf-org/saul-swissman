# Catalog — the filename↔law map

> Generated by the REGISTER ritual from corpus frontmatter. One row per **named** federal law (those whose
> frontmatter carries a `short_title`); 1184 of 5789 files **at this seed build** — the count drifts as
> legalize-ch updates upstream, so re-run REGISTER after a clone/pull to reconcile it. This is the
> human-browsable map (sorted by SR); for routing, **grep `locator.tsv`** — one TAB line per law with the
> file and the present-article ranges — and grep this catalog only to discover an abbrev by title. Never
> Read either file whole. The 4605 unnamed files (technical ordinances, treaties) are intentionally absent —
> a question about one of them is **not covered**; say so and point to Fedlex. Never grep the corpus as a whole.
>
> `seed` = vendored in this repo (works offline). Everything else needs `./launch.sh` to clone legalize-ch;
> until then `/api/article` and Read return *law not in local corpus*. `arts` = heading-anchored articles
> present; **a low `arts` or `max` far below the law's true size means the local copy is truncated** — verify
> per article before quoting (Law 4). `Stand` is the consolidation date, not proof no later amendment exists.
